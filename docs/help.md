`box help`
==========

Display help for the Box CLI

* [`box help [COMMAND]`](#box-help-command)

## `box help [COMMAND]`

display help for box

```
USAGE
  $ box help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
